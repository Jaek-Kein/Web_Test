Search API


Naver search results for each field, such as the web, news, and blogs, can be displayed directly on the web service or mobile app. In addition, you can search for regions such as "OO Station Restaurant", and additionally, you can use the function of identifying adult search terms and converting Korean-English key typos.


Naver Search Result Content

You can show the search results for Naver Web Service or mobile app of Web Document/Blog/News/Book/Movie/Cafe Post/Image/Encyclopedia data fields.

Region Search

When you want to show the search results such as "OO Station" and "O-dong bar" is used to show the results of each local services registered in Naver local services registered in Naver local services.

Search add-ons

As an additional search function, it provides a function that informs whether a specific search word is an adult search word or not, and a typo conversion function that corrects typos entered in the search box.

* Usage Limit : 25,000 per day
